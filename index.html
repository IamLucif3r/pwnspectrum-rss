gospray - Simple LDAP bind-based password spray tool

gospray is a low-profile, Go-based password spray tool leveraging LDAP bind authentication against domain controllers. The tool facilitates concurrent credential attempts by utilizing configurable Go routines. It operates without reliance on SMB or other intermediary servers, minimizing network footprint and operational noise. This technique can be employed for reconnaissance and credential stuffing activities.

Samsungs image library flaw opens a zero-click backdoor

Samsung has disclosed a critical vulnerability, CVE-2025-21043, impacting libimagecodec.quram.so, a Samsung-specific image parsing library used across Android 13-16 devices. This out-of-bounds write vulnerability allows for remote code execution (RCE) via specially crafted image files. Meta/WhatsApp privately reported the issue in August, highlighting messaging apps as a potential vector. The CVSS score of 8.8 indicates a high severity. Without patching, the flaw poses a significant risk, potentially enabling access to sensitive business apps, email accounts, and corporate data, especially within BYOD or mixed-managed environments. Organizations need to prioritize patching and implement robust mobile device management (MDM) strategies to mitigate this risk.

Source: https://www.csoonline.com/feed | Published: 15 Sep 2025

Cursor IDE: Arbitrary Data Exfiltration Via Mermaid (CVE-2025-54132)

A critical vulnerability (CVE-2025-54132) was identified within the Cursor IDE, impacting its ability to render Mermaid diagrams. This flaw allows for unauthorized data exfiltration via the manipulation of the diagram's underlying graph data structure. Specifically, malicious actors can leverage the diagram rendering process to inject and subsequently exfiltrate sensitive information. Remediation involved a fix targeting the parsing of the Mermaid syntax and sanitization of the generated graph data, effectively mitigating the risk of this graph-based data exfiltration vector.
