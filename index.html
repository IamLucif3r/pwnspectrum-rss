gospray - Simple LDAP bind-based password spray tool

gospray is a lightweight, Go-based LDAP bind-based password spraying tool. It performs authenticated LDAP queries directly against domain controllers to brute-force credentials. The tool’s architecture facilitates configurable concurrency via Go routines, minimizing network footprint and operational noise. Its design eliminates reliance on dependent services like SMB servers as targets, enabling stealthy credential enumeration.

Samsungs image library flaw opens a zero-click backdoor

Samsung has disclosed a critical vulnerability, CVE-2025-21043, impacting its Android devices (versions 13-16) via a zero-day exploit within the libimagecodec.quram.so library. This out-of-bounds write vulnerability enables remote code execution (RCE) through crafted image files. The flaw was privately reported to Samsung by Meta/WhatsApp in August and has implications for KNOX-protected environments, potentially bypassing device management controls. The vulnerability’s prevalence across system apps (Gallery, Camera) and third-party apps highlights the risk, particularly in BYOD or mixed-managed environments where patch compliance can be difficult to track. Remediation requires SMR Sep-2025 Release 1.
